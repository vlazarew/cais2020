Программа написана на Python 3.7.
Присутствуют 2 аргумента запуска --input и --output.
Запускать следует следующей командой "python3 main.py --input 01-linear.json --output test.dot" (Пример для первыйх открытых тестов. В input надо подставить необходимую вам json. В output будет записываться результат).
Генерация картинки выполняется командой "dot -Tpng -O output.dot"
